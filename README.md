# 🌤️ Dynamic Weather App

A beautiful, real-time weather application with a modern glassmorphism UI. The app focuses on providing accurate current weather conditions for any city around the world, paired with a dynamic background that changes to match the weather outside.

**🔗 [View Live Preview Here](#)**

## ✨ Features
* **Real-Time Weather Data:** Get accurate, up-to-the-minute current conditions and temperatures. 
* **City Search:** Easily search for current weather details in any city globally.
* **Unit Toggle:** Seamlessly switch between Metric (Celsius, km/h) and Imperial (Fahrenheit, mph) measurements with a click.
* **Dynamic Backgrounds:** The background image updates automatically to visually match the current weather (e.g., rainy, sunny, cloudy) using the Pexels API.
* **Modern UI:** A clean, glassmorphic design generated with the help of Google Stitch AI.

## 🛠️ Built With
* **HTML/CSS:** For structure and styling (Glassmorphism design).
* **Vanilla JavaScript:** For logic and DOM manipulation.
* **Visual Crossing API:** For fetching reliable, real-time weather data.
* **Pexels API:** For fetching high-quality, weather-matching background images.
* **Google Stitch AI:** Used for initial UI design and layout generation.

## 🧠 What I Learned
Building this project was a massive step forward in my JavaScript journey. Specifically, it helped me master:
* **Asynchronous JavaScript:** I learned how JavaScript handles operations that take time (like fetching data from a server) without freezing the rest of the website.
* **Promises & Async/Await:** I got hands-on experience writing clean, modern asynchronous code using `async` and `await` to pause my functions until the API data arrived.
* **Working with APIs:** I learned how to read API documentation, structure `fetch()` requests, handle JSON responses, and chain multiple APIs together (using the weather condition from one API to search for an image in another).
* **DOM Manipulation:** Safely taking data from an API and injecting it into the HTML structure to create a dynamic user interface.
